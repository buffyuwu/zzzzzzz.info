This repository, zzzzzzz.info, is a spiritual successor to the closed source yyyyyyy.info project which was made by artist Michael Guidetti. yyyyyyy.info was discontinued in 2025.

It contains images largely captured from the gifcities archive from lost/offline websites from the early 00's. All images belong to their respective owners... If said owners/entities still exist.

Made for The Final Nights community's in-game phone web browser (wwww.thefinalnights.com) so there are a lot of World of Darkness-esque references if you look close enough

RIP yyyyyyy.info
